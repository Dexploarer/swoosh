# Slash Commands

These command prompts are mirrored into `.claude/commands/` for Claude Code-style use. Other agents can paste them manually.

Recommended cadence:

1. `/apple-plan`
2. `/apple-fruta-architect` if architecture/targets are involved
3. `/apple-agent-tool` or `/apple-app-intent` for implementation
4. `/apple-test`
5. `/apple-privacy-review`
6. `/apple-release`
