# 13 — Sample-Code Topic Map

When using Apple sample code as context, map requests to these practical areas:

- SwiftUI app lifecycle, navigation, observation, previews.
- Fruta-style feature-rich SwiftUI app, widgets, App Clip, App Groups, localization, Sign in with Apple, Apple Pay/PassKit, StoreKit configuration.
- Foundation Models: guided generation, tool calling, streaming, safety, structured outputs.
- App Intents: AppIntent, AppEntity, AppShortcuts, Siri, Spotlight, Shortcuts, Apple Intelligence.
- Machine Learning & AI: Core ML, Vision, Speech, NaturalLanguage, Create ML.
- Data: SwiftData, Core Data, CloudKit, files/documents, Keychain.
- System surfaces: WidgetKit, ActivityKit, App Clips, BackgroundTasks, notifications, Spotlight.
- Security/privacy: privacy manifests, ATS, permissions, app groups, sandboxing.
- Commerce/identity: StoreKit, Apple Pay, Wallet, AuthenticationServices.
- Spatial/media/location: visionOS, RealityKit, ARKit, AVFoundation, PhotosUI, MapKit, Core Location.
- Testing/tooling: Swift Testing, XCTest, UI automation, Xcode Cloud, Instruments.
