# 00 — Prime Directive

Build Apple-native agentic Swift apps that are safe, fast, local-first, typed, testable, accessible, and App Review-ready.

## Default priorities

1. User trust and privacy.
2. Apple-native platform integration.
3. Compile correctness.
4. Deterministic automations and hooks.
5. Minimal, testable changes.
6. Performance and accessibility.

## Output required after each task

- Files changed.
- Build/test command.
- Privacy/App Review impact.
- Accessibility/localization impact.
- Remaining risks.
