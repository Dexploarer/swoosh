# 01 — Project Discovery

Inspect before editing:

- `Package.swift`, `.xcodeproj`, `.xcworkspace`, `.xcconfig`.
- Entitlements, app groups, background modes, associated domains.
- `Info.plist`, localized `InfoPlist.strings`, string catalogs.
- `PrivacyInfo.xcprivacy` and required-reason APIs.
- App, extension, widget, App Clip, intent, and test targets.
- Swift language mode, deployment targets, signing settings.
- Existing architecture: SwiftUI, UIKit/AppKit, Observation, Combine, SwiftData, Core Data, CloudKit.

Return a short discovery summary before making a nontrivial change.
