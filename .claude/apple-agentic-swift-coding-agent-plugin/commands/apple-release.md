# /apple-release

Prepare for TestFlight/App Store submission.

Verify:

- Archive/build passes.
- Privacy manifest/App Privacy labels/permission strings aligned.
- App Review notes include AI behavior, model location, test accounts, feature flags, payments, App Clip invocation, shortcuts, and sensitive permissions.
- Widgets/App Clip/Intents/Live Activities smoke-tested.
- Localization, accessibility, crash logs, StoreKit, Sign in with Apple, Apple Pay checked where relevant.
- No debug secrets or broad ATS exceptions.
